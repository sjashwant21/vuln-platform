# Security Policy

## Supported Versions

The VulnAssess Platform provides security updates for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

We take the security of VulnAssess seriously. If you believe you have found a security vulnerability in this project, please report it to us directly.

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please send an email to **shaanjashwant21@gmail.com**.

Please include the following information in your report:
- Type of vulnerability (e.g., XSS, SQLi, CSRF, etc.)
- Steps to reproduce the issue
- Proof of concept (PoC) code or screenshots
- Potential impact

We will acknowledge receipt of your vulnerability report within 48 hours and strive to provide a resolution or update within 7 days. Once the issue is resolved, we will publish a security advisory and notify you.
